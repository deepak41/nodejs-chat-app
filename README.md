# NodeJS Chat App


NodeJs, ExpressJs & Socket.io based real-time chat service.


## Getting Started
### To run the project:  
1. Clone the project  
`$ git clone https://github.com/deepak41/nodejs-chat-app.git`


2. Then enter the following commands:   
`cd nodejs-chat-app`  
`npm i`  
`npm run dev`  

The app should be up and running. 

## Getting Started
### To provide port no.:
`npm run dev port 8001`
