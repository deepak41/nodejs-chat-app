# Nodejs Chat App


NodeJs, ExpressJs & Socket.io based chat service.


## Getting Started
### To run the project:  
1. Clone the project  
`$ git clone https://github.com/deepak41/nodejs-chat-app.git`


2. Then enter the following commands:   
`$ cd nodejs-chat-app`  
`$ sudo npm install`  
`$ sudo nodemon`  

The app should be up and running.
